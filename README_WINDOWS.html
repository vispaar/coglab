This CD includes versions of CogLab On A CD for the following operating systems:

  • MacOS X
  • Windows

CogLab On A CD does not install anything on your computer.

If you use this disk on an Apple Macintosh Computer, you will see only the version for MacOS. If you use this disk on a computer running Microsoft Windows, you will see only the version for Windows.

For late breaking information, please see http://psychology.wadsworth.com/CogLabCD/

If you are running Windows, CogLab On A CD should automatically begin running when you insert the CD. If it does not, please double-click on My Computer on your desktop, then double-click on the CD-ROM icon, then double-click on the COGLAB application.

If you want to run CogLab On A CD for MacOS X, please insert the CD into an Apple Macintosh computer and read the "README_MAC.html" document.

Known Issues

  • CogLab On A CD will display a warning message if you try to close a demonstration window and have not yet saved the data. However, if you close the main window, you will not see the warning message: The program will immediately quit and you will lose your unsaved data.

  • Because CogLab On A CD is designed to be a cross-platform program, you cannot open a saved .cgl file by double-clicking on it. Instead, start the CogLab On A CD program and then select the "Open data in CogLab format" menu option under the "File" menu in the main window.

  • Some early versions of Windows have limits on the length of the file name. If you have a particularly long name, and are saving a lab that also has a long name (e.g., Apparent motion), you should verify that the file name is short enough so that the final .cgl, .html, or .txt has not been truncated by the computer.

  • Some labs may no longer display information correctly if you resize the window in which the experiment is running.
